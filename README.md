# Hotel-Management c++
The Hotel Management System is a simple console-based application developed using C++.
This project is designed to manage basic hotel operations such as room booking, customer details management, and rent calculation.
It demonstrates the use of Object-Oriented Programming (OOP) concepts like classes, objects, functions, and menu-driven programs in C++.
This project is suitable for beginner programmers and BCA students to understand real-world application development using C++
